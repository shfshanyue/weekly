# 前端周刊 (fe-weekly)

每周更新的前端技术周刊，基于 [Astro](https://astro.build) 构建。

## 快速开始

```bash
pnpm install       # 安装依赖
pnpm dev           # 启动开发服务器 (localhost:4321)
pnpm build         # 构建生产版本
pnpm preview       # 本地预览构建产物
```
