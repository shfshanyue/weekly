# 参与周刊

``` bash
# 新建一个新的周刊，比如第 100 期
$ vim docs/week-100.yaml

# 将第 100 期周刊生成 markdown 内容
$ pnpx tsx scripts/week.ts --week 100

# 或者将 1-100 期周刊生成 markdown 内容
$ seq 1 100 | xargs -I {} pnpx tsx scripts/week.ts --week {}

# 进行开发模式，查看效果
$ npm run dev
```
